### Твоя Git-шпаргалка (обновлённая)

**Просмотр состояния**  
```bash
git status
```

**Добавить все новые/изменённые файлы**  
```bash
git add .
```

**Сохранить изменения локально (коммит)**  
```bash
git commit -m "описание того, что сделано"
```

**Отправить коммиты на GitHub**  
```bash
git push
```

**Забрать последнюю версию с GitHub (чтобы получить изменения от нейросети или с другого компьютера)**  
```bash
git pull
```
> Важно: перед `pull` обязательно сохрани свои локальные изменения через `commit`, иначе могут возникнуть конфликты.

**Посмотреть историю коммитов**  
```bash
git log
```
